
## MOBI

모비는 교육기관에서 수업을 듣기에는 금액적, 시간적으로 부담스럽거나 이전에 사전 학습을 통해 실무를 경험하고 싶은 대학생이 최소한의 비용과 작은 부담으로 입문할 수 있도록 실무진이 멘토링 해주며 올바른 방향성을 제시하는 단체입니다. 모비의 수익금은 최소한의 운영비를 제외하고 소외 계층과 어려운 이웃에게 기부하고 있습니다.
